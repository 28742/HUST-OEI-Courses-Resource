## 前言

**Why？**:fireworks:

当今社会我们面临的普遍矛盾：越来越多的东西需要学习，时间却越来越少。

因此，我们希望可以建立一个针对HUST-EIC的课程共享资料库，帮助大家提升课程的学习效率。

同时，我们也希望提供一些方向的学习路线与课外资源，帮助后面的同学可以更早的看清前进的方向，在努力的路上少踩一些坑。
